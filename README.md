# Netflix ML Classification Project

## Objective
To analyze Netflix dataset and build ML models to predict content type.

## Techniques Used
- EDA
- Hypothesis Testing
- Feature Engineering
- NLP Processing
- Logistic Regression
- Random Forest
- XGBoost
- Hyperparameter Tuning
- Model Deployment using Joblib

## Best Model
XGBoost (Highest F1-score & Accuracy)

## Deployment
Model saved as pickle file and tested on unseen data.
